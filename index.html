easy_agile by SphereConsultingInc

Easy Agile is a Redmine plugin for agile project management.

Dependencies

gruff

Install

Install Redmine. Follow the Redmine plugin installation manual. Or go this way (inside the Redmine directory): ./script/plugin install git@github.com:SphereConsultingInc/easy_agile.git rake db:migrate_plugins RAILS_ENV=production (or simply "rake db:migrate_plugins" for development mode) Install nested_layouts plugin (from Redmine root directory): ./script/plugin install git://github.com/jwigal/nested_layouts.git Install gruffs library (if you want your Redmine installation to be portable): gem install gruff cd vendor/plugins && gem unpack gruff or (if you want to use gruff from gems): gem install gruff and add the following line to your config/environment.rb file: config.gem "gruff" Then restart Redmine. In project’s setting in Modules include ‘Easy Agile’ or enable it globally for all projects from Administration area. It will be available under ‘Easy Agile’ tab.
